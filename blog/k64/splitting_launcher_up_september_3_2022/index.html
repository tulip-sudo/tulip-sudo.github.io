Splitting Launcher Up

I have decided to split up the launcher into two seperate projects.
The launcher itself is the first project and only will have slight modifications.
The second project is Library Super Mario Sixty Four PC Launcher Backend (lib name: libsmsfpclauncherbackend).
This will be a changed version of the Kirigami64 backend (which stored important information about builds).
Hopefully implementing this won't take too long.
